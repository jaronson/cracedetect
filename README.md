# cracedetect
cracedetect is a live face detection & recognition application using OpenCV.

# Building
* `mkdir build`
* `cd build && cmake .. && make`

# Development

* `npm install -g grunt`
* `npm install -g grunt-cli`
* `npm install --save-dev`
* `git clone https://github.com/jaronson/cpplint`
* `cd cpplint && python setup.py install`
